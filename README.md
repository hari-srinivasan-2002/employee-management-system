 Employee Management System

A full-stack Employee Management System developed using React, Spring Boot, and MySQL.

 Features

* User Login Authentication
* Employee CRUD Operations

  * Add Employee
  * View Employee
  * Update Employee
  * Delete Employee
* Employee Search
* Dashboard Statistics

* 
* Department Analytics Chart
* Export Employee Data to Excel
* Responsive Navigation Bar

 Technologies Used

 Frontend

* React.js
* React Router DOM
* Axios
* Recharts
* XLSX
* File Saver

 Backend

* Spring Boot
* Spring Data JPA
* REST API
* Hibernate

 Database

* MySQL

 Project Structure

 Installation

 Backend

1. Configure MySQL database.
2. Update application.properties.
3. Run Spring Boot application.

 Frontend

npm install
npm run dev
